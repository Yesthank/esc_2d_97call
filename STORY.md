## 방 구성
- 한 방 = 최대 2시점 (앞/뒤 또는 좌/우)
- 문제가 많으면 방을 추가 (문/열쇠로 이동)
- 퍼즐 2~3개 풀면 다음 방 잠금 해제

| 방 | 시점1 테마 | 시점2 테마 | 퍼즐 수 | 다음 방 이동 조건 |
|----|----------|----------|--------|---------------|
| 방1 | | | | |
| 방2 | | | | |
| 방3 | | | | |

### 방 이동 흐름 예시
```
방1 (퍼즐 2~3개) → 열쇠/비밀번호 → 방2 (퍼즐 2~3개) → 열쇠 → 방3 → 탈출
```
