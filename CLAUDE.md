# CLAUDE.md

## 프로젝트 개요

**응답하라 97년생**
2D 포인트앤클릭 방탈출 게임. 코믹 테마. React + TypeScript + Vite 기반 예정. 모바일 퍼스트(세로 화면 비율, 데스크탑에서는 유튜브 숏폼처럼 가운데 정렬).

- 7개 방, 시점 없음 (방당 단일 화면)
- 10문제 내외 (방당 1~2문제)
- 문제 넘버링: `문제{방번호}-{순번}` (예: 문제2-1, 문제2-2)
- 배경: 나노바나나(AI 일러스트 생성) 활용 + 게임 캡처/프롬프트 제작
- 퍼즐 일부도 일러스트로 제작

## 스토리 요약

97년생 고3 학생 **욱광민**이 야간자율학습을 째고 PC방에 가려다, 담임 **금영기**에게 잡힘.
분노한 금영기가 욱광민을 게임 속 세계로 보내버림.
추억의 온라인 게임들을 돌며 퍼즐을 풀고 현실로 돌아와야 하는 좌충우돌 모험.
**반전**: 사실 전부 야자 중 엎드려 잔 꿈이었고, 금영기에게 또 걸림.

| 방 | 장소 | 분위기 | 퍼즐 수 | 배경 | 비고 |
|----|------|--------|---------|------|------|
| 방1 | 교실 (야자시간) | 현실/코믹 | 1 | 일러스트 | 탈출 경로 퍼즐 |
| 방2 | 복도 | 현실/코믹 | 1 | 일러스트 | 교실 창문 단서 (검토중) |

| 방4 | 피파온라인 | 게임 세계 | 1~2 | 게임 캡처/일러스트 | NPC 터치 → 문제 |
| 방5 | 서든어택 | 게임 세계 | 1~2 | 게임 캡처/일러스트 | NPC 터치 → 문제 |
| 방6 | 던전앤파이터 | 게임 세계 | 1~2 | 게임 캡처/일러스트 | NPC 터치 → 문제 |
| 방7 | 리그오브레전드 | 게임 세계 | 1~2 | 게임 캡처/일러스트 | NPC 터치 → 문제 |

### 방 이동 흐름

```
방1 교실 → 방2 복도 → (금영기 등장, 게임 속으로 워프)
→ 방3 메이플스토리 → 방4 피파온라인 → 방5 서든어택
→ 방6 던전앤파이터 → 방7 리그오브레전드
→ 엔딩 (꿈이었다 + 방1 교실 재등장 + 금영기 대사)
```

## 방별 상세

### 방1 - 교실
- **상황**: 야간자율학습 시간, 친구와 PC방 가기로 작당
- **배경(일러스트)**: 교실 전경. 자습 중인 학생 뒷모습, 졸고 있는 금영기, 우측 창문들, 철문
- **문제1-1**: 선생님께 걸리지 않고 교실 탈출하는 최소 경로 (상세 미정)

### 방2 - 복도
- **상황**: 교실 탈출 후 안심하는 짧은 대사
- **배경(일러스트)**: 가운데 길게 뻗은 복도, 끝에 출구, 양쪽 불 켜진 교실들
- **문제2-1**: 복도 양쪽 교실 창문을 단서로 활용 (상세 미정)
- **문제 클리어 후(일러스트)**: 밤공기와 달을 보며 기분 좋아하는 장면 → 금영기 등장 → 대사 후 게임 세계로 워프

### 방3~7 - 게임 세계
- 각 게임의 추억 소재를 퍼즐로 활용
- NPC/몬스터를 터치/클릭하여 문제 진입
- 배경은 실제 게임 화면 캡처 또는 프롬프트로 유사하게 제작
- 상세 퍼즐 내용 미정

### 엔딩
- 모든 시련 극복 후 암전
- 맨 처음 방1 교실이 다시 등장
- 금영기가 욱광민을 바라보고 있음
- **대사**: "이래도 꿈이가, 욱광민은 맨날 잡니다. 앞으로 나와!"

## 일러스트 관리

- `*일러스트*`로 표기된 항목은 나노바나나로 제작
- 생성된 일러스트는 지정된 폴더/파일명으로 저장 (폴더 구조는 개발 착수 시 확정)
- 배경 일러스트: 각 방의 메인 배경
- 퍼즐 일러스트: 문제 자체가 이미지인 경우
- 컷씬 일러스트: 방 이동/이벤트 연출용
