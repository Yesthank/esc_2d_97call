# 일러스트 제작 가이드

> 나노바나나 프롬프트 작성 시 이 문서를 참고하여 일관된 스타일을 유지한다.

---

## 공통 스타일 프롬프트 (모든 일러스트에 적용)

아래 베이스 프롬프트를 모든 장면 프롬프트 앞에 붙여 사용한다.

```
Korean cartoon/comic style, bright vivid colors, bold outlines, exaggerated expressions,
comedic tone, flat shading with minimal gradients, clean vector-like illustration,
vertical mobile aspect ratio (9:16)
```

### 스타일 키워드 정리

| 항목 | 설정 |
|------|------|
| 화풍 | 카툰/코믹 (한국 웹툰 느낌) |
| 선 | 굵고 깔끔한 외곽선 (bold outline) |
| 색감 | 밝고 선명한 색상, 과장된 표현 |
| 셰이딩 | 플랫 셰이딩, 최소한의 그라디언트 |
| 톤 | 코믹/유머러스 |
| 비율 | 세로 9:16 (모바일 세로 화면) |

### 캐릭터 설정

| 캐릭터 | 외형 키워드 |
|--------|------------|
| 욱광민 (주인공) | 남고생, 교복, 짧은 머리, 장난기 있는 표정 |
| 금영기 (담임) | 남성 교사, 안경, 근엄한 표정 (화나면 불그락) |
| 급우들 | 교복 차림의 다양한 남학생들 |

## 사용법

1. 장면이 필요할 때 이 문서에서 해당 항목을 찾는다
2. **공통 스타일 프롬프트** + 개별 장면 프롬프트를 합쳐 나노바나나에 입력
3. 콘티(스케치/레이아웃)가 제공된 경우, 콘티와 동일한 화면 비율로 제작한다
4. 생성된 이미지를 지정된 파일명으로 해당 폴더에 저장
5. 상태를 `완료`로 업데이트
